# Firebase Authentication App

## 📱 Features
- User Signup & Login
- Firebase Authentication
- Home Screen Navigation
- Logout Functionality
- Auto Login

## 🛠 Technologies Used
- Android Studio
- Kotlin
- Firebase

## 🚀 Description
This app allows users to register and login using Firebase Authentication. 
After login, users are redirected to a home screen where they can logout. 
The app also supports auto-login for better user experience.
